# Проект: прогноз оттока клиентов

Это учебный проект по задаче телеком-оператора: по данным о договорах, клиентах и подключенных услугах нужно предсказать, разорвет ли клиент договор.

В проекте:
- загружаются данные из `ds-plus-final.db`;
- выполняются EDA и предобработка;
- обучаются несколько моделей классификации;
- выбирается лучшая модель по ROC-AUC и проводится тестирование;
- анализируется важность признаков.

Как запустить зависимости из `requirements.txt`:
1. Откройте терминал в папке проекта.
2. Создайте виртуальное окружение:
   `python3 -m venv .venv`
3. Активируйте окружение:
   `source .venv/bin/activate`
4. Установите зависимости:
   `pip install -r requirements.txt`

После установки зависимостей откройте и запустите `telecom_churn_notebook.ipynb`.
